How to Create a Case for SE Support

Currently, Worldwide System Engineers have 3 active options to raise a "SE Support" tickets.

Option 1: Using the SFDC Case Object

Introducing the new SE Support Ticketing system available under the customer/prospect Account object in SFDC. SFDC case creation allows SE to add all the relevant information into the ticket before submitting it to ensure quick turnaround.

Use the following steps to successfully submit a "SE Support" Ticket in SFDC:

1. In Salesforce, navigate to a Customer/Prospect/Channel Account.

2. Click on 'Cases'.

3. Click on 'New Case' to open the form, and select 'SE Support' as the record type. Click Next to proceed.

4. Using the Type drop-down, select the service from the drop-down list.

5. Fill in all the mandatory information with the red asterisks and click Save.

6. Attachment: Attach all the relevant documents under "Related" tab once you save the ticket, this will avoid delays in the resolution.

Option 2: Using the Helper App

Use the following steps to successfully submit a "SE Support" Ticket in the Helper App:

  1. Login to https://helper.nutanix.com/Account.
  2. Select your account under which you need to create a "Case".
  3. Click on 'Cases' > 'New'.
  4. Fill the form with the mandatory information and click Save. Once its saved, a "SE Support" ticket will be automatically created under the account.

Option 3: Using SE GPT

Follow the below steps to raise a SE Support ticket via SE GPT:

1. While querying in the SE GPT, scroll to the extreme bottom of your query response and click on "Create SE Support Case".

2. Search the account under which you need to raise a SE Support Ticket.

3. Click "Next" and fill in the form. Select the Type (Service Type), attach any relevant document before hitting the "Submit" button.
